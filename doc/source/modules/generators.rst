Generator Modules
=================

Generator modules produce noise without requiring a source to draw on. They can be used to output values in their own right, or combined and transformed using the modifier modules. This page contains documentation for these modules generated from the docstrings of the classes themselves, which are in turn ported from the original in-code documentation of libnoise (albeit with some modifications).
